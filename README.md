# Quick Env Switcher

## Overview
Quick Env Switcher is a tool designed to simplify the management of multiple environment configurations for web developers and small teams. With a single command, you can seamlessly switch between different project environments.

## Features
- Manage environment variables easily.
- User-friendly prompts for selecting environments.
- Support for custom user-defined profiles.
- Integration with CI/CD tools for automated switching.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/finggu-quick-env-switcher.git
   cd finggu-quick-env-switcher
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   node src/index.js
   ```

## Usage
Follow the prompts to select and switch between your configured environments.
